# Python-weather-app-
  Python Weather App (PyQt5)

## Features

✔ Real-time weather data  
✔ City-based search  
✔ Temperature in Celsius  
✔ Weather description display  
✔ Error handling for invalid city names  
✔ Clean and responsive GUI  

## Tech Stack

- Python
- PyQt5
- Requests
- OpenWeatherMap API
